# Fintech AI Assistant App

A comprehensive AI-driven personal finance management application designed to help users track expenses, manage income, and gain financial insights using machine learning.

## 🚀 Features

- **AI Expense Prediction**: Predicts future spending patterns based on historical data.
- **OCR Receipt Scanning**: Automatically extract expense details from receipts using Google ML Kit.
- **Voice-Enabled Commands**: Add expenses and query financial status using voice.
- **Financial Analytics**: Visualize spending trends with interactive charts (MPAndroidChart).
- **Secure Authentication**: User signup/login with biometric security support.
- **Real-time Sync**: Synchronize local data with a Python-based backend.
- **Intelligent Insights**: Personalized financial advice generated by AI.

## 🛠 Technology Stack

### Android Application
- **Language**: Java
- **UI Framework**: Material Design Components
- **Local Database**: Room Persistence Library (SQLite)
- **Networking**: Retrofit & OkHttp
- **ML & AI**: Google ML Kit (OCR), Custom AI Integration
- **Charts**: MPAndroidChart
- **Background Tasks**: WorkManager

### Backend Service
- **Language**: Python
- **Libraries**: Flask/FastAPI, Scikit-learn (ML), Pandas
- **Logic**: NLP parsing, spending prediction models, data analytics

## 🏗 Architecture

The project follows a **Repository-based Architecture** (inspired by MVVM) which ensures separation of concerns:
- **UI Layer**: Activities and Fragments handle user interaction.
- **Data Layer**: Repositories manage data flow between Room (local) and Retrofit (remote).
- **Service Layer**: Background services and WorkManager handle synchronization and long-running tasks.

## ⚙️ Installation & Build

1. **Android App**:
   - Open the `Android_App` folder in Android Studio.
   - Sync Gradle and build the project.
   - Ensure you have an Android device/emulator with API level 24+.

2. **Backend**:
   - Navigate to `Backend/Source_Code`.
   - Install dependencies: `pip install -r requirements.txt`.
   - Run the server: `python app.py`.

Refer to [Documentation/Installation_Guide/INSTALLATION.md](Documentation/Installation_Guide/INSTALLATION.md) for detailed steps.
